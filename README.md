# Análise de Vendas App

## Aplicativo Android

Aplicativo desenvolvido em Java no Android Studio com o objetivo de realizar cadastro e análise de produtos, permitindo visualizar informações importantes para tomada de decisão.

---

## Funcionalidades

- Cadastro de produtos com:
  - Nome
  - Categoria
  - Preço de compra
  - Preço de venda
- Cálculo automático de lucro
- Armazenamento local dos dados (SharedPreferences)
- Listagem de produtos organizados por categoria
- Dashboard com análise de vendas:
  - Lucro total
  - Margem de lucro
  - Produto mais lucrativo
  - Produto com menor lucro
  - Ranking (Top 3 produtos)
  - Categoria mais utilizada

---

## Como testar

1. Abra o aplicativo
2. Cadastre um produto
3. Acesse a tela "Ver produtos cadastrados"
4. Acesse a tela "Análise de Produtos"
5. Visualize os dados calculados automaticamente
